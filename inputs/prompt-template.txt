RESEARCH DEPTH (optional — drives `RESEARCH_MODE` if env not set)
----------------------------------------------------------------
Add **one** line anywhere in the prompt:

  **Research depth:** `full` | `gapfill` | `messaging` | `skip`

  • **full** — broad AskBill + Glean (12–18+ tool calls); use when you need heavy Gong/collateral
    or the skill is missing/outdated for your products.
  • **gapfill** — **pipeline default** when this line is omitted; targeted AskBill for API gaps,
    minimal Glean unless needed (set explicitly or rely on default).
  • **messaging** — prioritize Glean for sales stories; few AskBill calls.
  • **skip** — no research agent; uses skill + curated KB only (fastest; verify messaging
    yourself). Falls back to **full** if the skill file is missing.

You can also set the environment variable **`RESEARCH_MODE`** to the same values.
