VALUE PROP / MESSAGING NOTES (GLEAN-ALIGNED, COMPLIANCE-SAFE)
-------------------------------------------------------------------------------

Use these angles in narration and on-screen copy:
- Approve more good users with less friction by adding cash-flow context when bureau data is inconclusive.
- Position bank linking as a second-look path that can increase spending power confidence.
- Use the same connected account for both decisioning context and downstream repayment setup where appropriate.
- Move Plaid earlier in the funnel, not only at repayment, to improve BNPL coverage.
- Emphasize one-platform consolidation themes (risk + monitoring + consumer-reporting workflows) without overclaiming.
- Reinforce lifecycle value after approval: ongoing account context can support line maintenance and tailored offers.

Language guardrails:
- Prefer: "can help", "supports", "designed for", "input to underwriting".
- Avoid guarantees of approval, loss reduction, or fixed conversion lift.
