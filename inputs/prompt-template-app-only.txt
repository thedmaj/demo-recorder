HOW TO RUN (APP-ONLY / NO-SLIDE)
--------------------------------
1. Save this as `inputs/prompt.txt`
2. Run the full pipeline:

   ```bash
   npm run demo
   ```

   App-only is the pipeline default — no env flags required.

3. From the dashboard, leave the "Include slides phase" checkbox in the Run
   Pipeline card unchecked. The dashboard default is also app-only.

4. Optional: add `Brand URL: https://...` near the top for `brand-extract`.

NOTES
-----
- App-only is the default everywhere. To opt into slides, use
  `npm run demo:with-slides` (or `npm run demo -- --with-slides`) on the CLI,
  or check the dashboard "Include slides phase" checkbox.
- The single canonical switch is `PIPELINE_WITH_SLIDES=true|false`. Legacy envs
  (`SCRIPT_ZERO_SLIDE`, `DEMO_MARKETING_SLIDE`, `BUILD_PHASE_SEQUENCE`,
  `BUILD_PHASE_SLIDES_ENABLED`) still work but are no longer required.
- Keep storyboard beats as host/link/insight only (no slide beats).
